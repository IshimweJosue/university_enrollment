## Key Features

- ✅ Full CRUD operations for Students, Professors, Courses
- ✅ Many-to-Many relationships with Enrollments & Grades
- ✅ REST API with DRF
- ✅ Search & Filtering
- ✅ File uploads (PDFs, Images)
- ✅ Production deployment (Nginx + Gunicorn)
- ✅ Automated database backups

## Technology Stack

Django 4.2 • DRF • MySQL • Nginx • Gunicorn • HTML5 • CSS3 • JavaScript

## Getting Started

```bash
git clone https://github.com/YOUR_USERNAME/university_enrollment.git
cd university_enrollment
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
